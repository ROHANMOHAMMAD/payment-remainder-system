🚀 Quick Start

Option 1: Docker (Recommended)

docker-compose up -d

Then access: http://localhost:3000

Option 2: Local Setup

  1. cd backend && npm install && npm start
  2. cd frontend && npm install && npm start
  3. Setup PostgreSQL database

🚀 How to Run

1. Using Docker (Easiest - All Platforms)

cd /path/to/project docker-compose up -d # Wait 30 seconds # Open: http://localhost:3000

2. Local Development (Windows/Mac/Linux)

# Terminal 1 - Backend cd backend npm install npm start # Terminal 2 - Frontend cd frontend npm install npm start # Terminal 3 - Database (if needed) psql -U postgres -d payment_system -f database/schema.sql

3. Windows Auto Setup

setup.bat # Double-click or run in terminal

💡 Pro Tips

  • Having issues? Check backend logs: docker-compose logs backend
  • Database error? Restart database: docker-compose restart postgres
  • Want to customize? Edit frontend/index.html for UI or backend routes for logic
  • Need production? Set production env vars and use docker push to registry
  • Add features? Extend API in backend/routes and update frontend
